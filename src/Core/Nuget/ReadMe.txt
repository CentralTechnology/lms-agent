OneTrueError.Client installation
=================================

Congratulations on taking the first step towards a more efficient exception handling.

Now you need to either download and install the open source server: https://github.com/gauffininteractive/OneTrueError.Server/issues
.. or create an account at https://onetrueerror.com.

Once done, log into the server and find configuration instructions.
(Or read the articles in our documentation: https://onetrueerror.com/documentation)
